# PRD: 素晴日「角色对话生成器」Dialogue Composer

> 项目：SubaHibi 16th Anniversary（素晴日 16 周年纪念站）
> 版本：v1.0
> 日期：2026-05-28

---

## 1. 概述

在纪念站中新增**角色对话生成器**栏目。用户选择喜欢的角色、背景，输入自定义对话文案，实时合成一张仿游戏内对话框的 VN 风格图片，并支持下载。

**核心体验：**「你与角色对话，你进入游戏。」

---

## 2. 目标

| 目标 | 说明 |
|------|------|
| 互动性 | 区别于纯浏览型栏目，让用户主动创作 |
| 复用素材 | 利用已有立绘、背景、角色数据 |
| 传播性 | 生成图片可下载分享，为站点引流 |
| 低门槛 | 无需设计能力，三步完成：选角色 → 写文案 → 下载 |

---

## 3. 用户流程

```
进入 /dialogue 页面
  │
  ├── 1. 选择角色
  │     从 9 位主角团中选择（卡片展示，可实时切换）
  │     └─ 水上由岐 / 间宫卓司 / 高岛柘榴 / 若槻镜 /
  │        若槻司 / 间宫羽咲 / 音无彩名 / 间宫皆守 / 木村
  │
  ├── 2. 选择背景
  │     从 12-15 张精选背景中选择（按氛围分组）
  │     └─ 晴日 / 黄昏 / 夜晚 / 幻想
  │
  ├── 3. 编写对话
  │     ├─ 角色名（默认所选角色名，可修改）
  │     ├─ 对话正文（多行，每行 ≤30 字，总计 ≤200 字）
  │     └─ 支持「——」「……」等特殊标点
  │
  ├── 4. 实时预览
  │     右侧/下方显示合成效果
  │     └─ 背景 → 立绘 → VN 对话框 → 文字
  │
  └── 5. 下载 / 重新编辑
        ├─ 点击下载 PNG（1280×720）
        └─ 返回修改任一参数
```

## 4. 功能需求

### 4.1 角色选择器 (`CharacterPicker.tsx`)

- 以卡片网格展示 9 位角色
- 卡片包含：角色立绘缩略图 + 角色名 + 身份标签（如"眺望世界之人"）
- 点击选中，高亮状态
- 选中后左侧/预览区立绘切换为该角色
- 数据源：`src/content/characters.ts`

### 4.2 背景选择器 (`BackgroundPicker.tsx`)

- 分类标签：晴日 / 黄昏 / 夜晚 / 幻想
- 每类 3-4 张缩略图，点击选中
- 背景列表：

### 4.3 对话编辑器 (`DialogueEditor.tsx`)

- **角色名输入框**：默认填充所选角色名，可自定义
- **对话正文文本框**：多行 textarea
  - 每行 ≤30 个中文字符
  - 总计 ≤200 字符
  - 实时显示字数统计
- **自动换行**：长句按每行最大字数自动断行
- **特殊格式**：支持「——」「…」「！」「？」

### 4.4 合成预览 (`PreviewCanvas.tsx`)

合成图采用 **16:9 横版（1280×720）**，层次从上到下：

```
┌─────────────────────────────────────┐
│        背景图片（全幅）               │
│                                     │
│           立绘（居中偏左/右）         │
│                                     │
│  ┌────────────────────────────────┐ │
│  │ 角色名                         │ │
│  │ ───────────────────────────── │ │
│  │ 对话文字                       │ │
│  │                               │ │
│  └────────────────────────────────┘ │
└─────────────────────────────────────┘
```

**对话框样式（CSS 实现，不依赖素材图片）：**

```
┌──────────────────────────────────────┐
│ 水上由岐                              │  ← 角色名区域（半透明黑底+白字）
│──────────────────────────────────────│
│ 世界是我所看到、触摸到、                 │  ← 正文区域（半透明黑底+白字）
│ 以及感受到的东西。                      │
│                                       │
└──────────────────────────────────────┘
```

- 对话框位于画面底部约 1/4 位置
- 宽度约画面 85%，水平居中
- 半透明黑色背景（rgba(0,0,0,0.7)）
- 角色名区域与正文区域用细线分隔
- 字体：Noto Serif SC（--font-title），白色
- 角色名字号稍小（16px），正文字号 18-20px
- 底部留白边距

### 4.5 下载功能

- 使用 `html-to-image` 库将预览 DOM 区域导出为 PNG
- 导出尺寸：1280×720
- 下载文件名：`subahibi_dialogue_{角色英文名}_{时间戳}.png`
- 下载按钮状态：idle → generating → done

### 4.6 响应式

- 桌面端：左侧操作区 + 右侧预览区（并排）
- 平板/手机：操作区在上，预览区在下（堆叠）
- 所有选择器支持触屏操作

## 5. 技术方案

### 5.1 路由与目录结构

```
src/app/dialogue/
  page.tsx          — 对话生成器主页面
  layout.tsx        — 页面布局

src/components/dialogue/
  CharacterPicker.tsx   — 角色选择器
  BackgroundPicker.tsx  — 背景选择器
  DialogueEditor.tsx    — 文案编辑器
  PreviewCanvas.tsx     — 合成预览 + 下载
  dialogue.css          — 对话框样式
```

### 5.2 依赖

```json
"html-to-image": "^1.11.0"
```

- `html-to-image`：将 DOM 节点导出为 PNG，比 html2canvas 更轻量（~5KB）

### 5.3 素材来源

| 类型 | 来源 |
|------|------|
| 角色立绘 | `C:\Users\张靖浩\Desktop\素晴日\立绘\` → 复制到 `public/assets/chara/` |
| 背景图片 | `C:\Users\张靖浩\Desktop\素晴日\background\` → 复制到 `public/assets/bg/` |
| 对话框 | **CSS 实现**（不依赖外部图片） |

### 5.4 合成实现方式

**方式：DOM 截图法**

1. 在页面中渲染一个隐藏的 1280×720 的 `div.preview-target`
2. 内部结构：`<img bg> → <img chara> → <div dialog-box>`
3. 用户操作实时更新这个隐藏容器的内容
4. 点击下载时，用 `html-to-image` 将此容器导出为 PNG

**优点：**
- 文字渲染清晰（抗锯齿好）
- 样式控制灵活（CSS 完全可控）
- 无需处理 Canvas 文字换行
